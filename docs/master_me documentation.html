master_me

sound optimisation plugin for live-streaming and podcasting

Introduction

With the first lockdowns in 2019, several real-life conferences were forced into the digital domain and sounded crappy. Therefore: master_me

Open Source and licensing

mater_me is GPL3

System requirements and compatibility

master_me is available on windows, macOS and linux.

Due to faust and its open architecture, it can possibly be compiled to a vast variety of target platforms.

Modules

Module: pre-processing

The ‘pre-processing’ module contains:
- a gain-slider to apply gain to the incoming signal before it hits master_me.

- a ‘mono’ switch

- phase switches for both left and right channels

- ‘stereo-correct’ switch (see tba)

Module: gate

A simple noise gate with the following parameters;

- threshold

- attack

- release

Module: leveler
